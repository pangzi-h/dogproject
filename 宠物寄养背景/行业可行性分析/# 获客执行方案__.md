## 二、小红书获客执行方案

### 账号设置

**昵称：** 边牧小院｜郑州宠物寄养

**简介模板：**

```
郑州两口子养了2只边牧5年
专注宠物寄养 | 24h监控透明化
爱狗专业靠谱 | 可约看场地
坐标郑州XX区 | 私信必回
```

**头像：** 两只边牧合影（清晰、可爱、专业感）

**背景图：** 场地全景或边牧大片

### 每日发布计划

**发布时间：**

- 早晨：8:00-9:00
- 中午：12:00-13:00
- 晚上：19:00-21:00（黄金时间）

**发布数量：** 每日至少1条，建议2条

### 内容发布SOP

```
Step 1：选择内容类型
□ 边牧日常（搞笑/温馨）
□ 养宠干货（知识类）
□ 寄养故事（案例类）
□ 服务介绍（引流类）

Step 2：准备素材
□ 照片：2-5张（高质量、光线好）
□ 或视频：15-60秒

Step 3：撰写文案
□ 标题：有吸引力
□ 正文：真实、有价值
□ 结尾：引导互动/私信

Step 4：添加标签
#宠物寄养 #郑州宠物寄养 #边牧 #养狗日记

Step 5：发布
□ 选择封面图
□ 添加位置（郑州XX区）
□ 定时发布/即时发布

Step 6：互动维护
□ 30分钟内回复评论
□ 引导私信
□ 私信引导加微信
```

### 爆款标题模板

**模板1：数字+痛点**

```
养狗5年，我踩过的3个坑
郑州宠物寄养，这4点要注意
边牧掉毛季，我用了这5招
```

**模板2：悬念+探秘**

```
这只狗来寄养时，我差点拒绝了...
养边牧5年，我终于放弃了...
那个把狗送来寄养的人，后来怎样了
```

**模板3：痛点+解决**

```
狗狗挑食怎么办？我找到答案了
宠物寄养踩雷？教你避坑
边牧拆家？可能你做错了这件事
```

### 评论区互动话术

| 评论类型 | 回复话术 |
|----------|----------|
| 咨询价格 | "已私~" 或 "私信您具体价格哈" |
| 夸赞狗狗 | "谢谢亲~我家两只都是戏精哈哈" |
| 问地址 | "坐标郑州XX区，私信发您定位~" |
| 问服务 | "家庭式寄养，24h监控，私信细聊~" |
| 表达兴趣 | "欢迎来玩~有问题随时问我" |

### 私信转化话术

**私信回复模板：**

```
第一条回复：
您好！感谢关注~ 我们是家庭式宠物寄养，
在郑州XX区，有独立场地，24小时监控。
小型犬60/天，中型犬80/天。
方便加微信详聊吗？我发您场地照片~

第二条（加微信后）：
[发送场地照片3-5张]
这是我们场地环境，
您可以先看看~
有任何问题随时问我

第三条（跟进）：
您家狗狗是什么品种呀？
打算什么时候寄养呢？
可以提前约时间来看场地
```

- 潜客：咨询过未成交

**二级标签（宠物）：**

- 中型犬

**每日发布：**

- 早晨（8:00-9:00）：边牧日常/寄养状态

## 五、线下获客执行方案

### 社区渗透

**步骤1：摸排周边**

- [ ] 列出3公里内所有小区
- [ ] 列出附近宠物店、宠物医院
- [ ] 找到物业联系方式

**摸排记录表：**

| 类型 | 名称 | 地址 | 距离 | 联系方式 | 备注 |
|------|------|------|------|----------|------|
| 小区 | | | | | |
| 小区 | | | | | |
| 宠物店 | | | | | |
| 宠物医院 | | | | | |

**步骤2：混入宠物群**

方法：

1. 问遛狗的人："您好，请问这个小区有宠物群吗？"
2. 问物业："您好，请问咱们小区有宠物交流群吗？"

**步骤3：建立信任（7天后推广）**

前7天：

- 不打广告
- 分享养宠知识
- 回答问题
- 分享边牧照片

第7天后（有人问寄养时）：

```
"哦对，我家附近有个小院做寄养的，
挺靠谱的。需要的话我推荐？"
（或直接说自己的）
"我们家也做寄养，就在XX那边，
您可以过来看看环境~"
```

### 宠物店合作

**合作洽谈流程：**

```
Step 1：准备物料
□ 名片
□ 宣传单

Step 2：上门拜访
话术：
"老板您好，我在附近做宠物寄养，
想跟您这边合作一下，
咱们互相推荐客户，
我给您返点，您看怎么样？"

Step 3：留下联系方式
□ 留名片
□ 加微信
□ 留宣传单

Step 4：跟进维护
□ 每月问候
□ 有客户推荐及时反馈
□ 节假日送礼
```

**合作返点方案：**

- 推荐客户成交：返首单金额的10-20%
- 或：双方客户互享优惠

## 六、转介绍获客方案

### 转介绍机制

**推荐人奖励：**

- 推荐成功：下次寄养8.5折
- 推荐3人以上：终身8折

**被推荐人优惠：**

- 首次寄养9折

### 转介绍话术

**寄养结束时：**

```
"这次寄养很顺利，XX很乖~
如果觉得满意，
可以帮忙推荐给有需要的朋友~
朋友来，给您和ta都优惠。"
```

**客户回访时：**

```
"XX最近还好吗？
对了，如果您身边有朋友需要寄养，
欢迎推荐哈，有优惠的~"
```
