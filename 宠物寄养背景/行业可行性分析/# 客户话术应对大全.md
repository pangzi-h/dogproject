# 客户话术应对大全

## 一、售前咨询话术

### 1. 价格咨询

```
【客户：这个狗粮多少钱？】

话术A（直接报价）：
"您好，伊芙琳狗粮根据规格不同价格不同：
- 5kg装：XX元（适合小型犬）
- 10kg装：XX元（适合中型犬，性价比高）
- 15kg装：XX元（适合大型犬，最划算）
您家狗狗多大呀？我帮您推荐合适的规格"

话术B（价值引导）：
"这款狗粮性价比很高，肉含量65%，无谷物配方
同等品质的某品牌要卖XX元
我们直播间专属价只要XX元
而且前10名下单还送零食
您家狗狗什么品种？我帮您看看适不适合"
```

### 2. 品质咨询

```
【客户：这个狗粮好不好？】

话术A（真实体验）：
"这款我家两只边牧吃了3个月
之前掉毛很严重，现在好多了
毛发也变得亮泽
（发边牧照片）
我自己也在用，不好不会推荐"

话术B（数据说话）：
"您可以看下配料表：
✅ 前3位全是肉（鸡肉、鱼肉、牛肉）
✅ 肉含量65%，蛋白质32%
✅ 无谷物、无诱食剂、无防腐剂
✅ 含Omega-3，对毛发好
这个配置在同等价位里算很能打的"

话术C（对比引导）：
"您现在喂的是什么狗粮呀？
我可以帮您对比一下
如果狗狗现在吃得挺好，其实不用换
如果遇到掉毛、软便这些问题，可以试试这款"
```

### 3. 适用性咨询

```
【客户：我家狗狗能吃吗？】

话术A（了解情况）：
"请问您家狗狗是什么品种？多大了？
有没有什么特殊情况？
比如肠胃敏感、过敏史等
我帮您判断适不适合"

话术B（针对性回答）：
┌────────────────────────────────────────────────────┐
│ 小型犬（泰迪、比熊等）：                            │
│ "这款颗粒稍大，小型犬可能需要泡软喂"              │
│                                                    │
│ 中型犬（边牧、柯基等）：                            │
│ "非常适合！这款就是为中大型犬设计的"              │
│                                                    │
│ 大型犬（金毛、拉布拉多等）：                        │
│ "很合适，而且大包装性价比更高"                    │
│                                                    │
│ 肠胃敏感：                                         │
│ "这款无谷物配方，对肠胃友好                        │
│ 建议先买小包装试吃，适应了再买大的"               │
└────────────────────────────────────────────────────┘
```

## 二、异议处理话术

### 1. 价格异议

```
【客户：有点贵啊】

话术A（价值对比）：
"我理解您的感受
但咱们可以算笔账：
这款10kg装XX元，够中型犬吃1个多月
平均每天只要X元
比一杯奶茶还便宜
但狗狗吃得好，少去医院，其实更省钱"

话术B（品质对比）：
"确实不便宜，但品质也在这
同等配置的进口粮要贵一倍
这款是国产优质粮，性价比很高
而且我自己也在用，品质放心"

话术C（促销引导）：
"今天直播间有活动
平时卖XX，今天只要XX
还送零食，算下来很划算
而且首次购买还有会员价，下次更便宜"
```

### 2. 信任异议

```
【客户：没听过这个牌子】

话术A（品牌介绍）：
"伊芙琳是近两年新起的国产品牌
虽然名气不如皇家、渴望
但品质不输它们，价格却便宜很多
主打性价比，很适合精打细算的铲屎官"

话术B（真实背书）：
"我理解您的顾虑
我自己养边牧5年，试过很多品牌
这款是我筛选后觉得性价比最高的
我自己两只边牧都在吃
您可以先买小包装试试，满意再回购"

话术C（保障承诺）：
"没听过很正常，新品牌嘛
但品质有保障：
✅ 正规厂家生产
✅ 有检测报告
✅ 不满意包退
您可以放心尝试"
```

### 3. 效果异议

```
【客户：吃了真的有效果吗？】

话术A（真实案例）：
"效果因狗而异，但大多数狗狗反馈都不错
（发客户好评截图）
这是之前客户的反馈
您可以参考一下"

话术B（合理预期）：
"狗粮不是药，效果需要时间
一般吃1-2周能看到便便改善
1-2个月能看到毛发变化
建议坚持吃一段时间才能看到效果"

话术C（试用建议）：
"建议您先买小包装试吃
看狗狗爱不爱吃、适应不适应
适应了再买大包装
这样最稳妥"
```

## 四、售后服务话术

### 1. 物流问题

```
【客户：怎么还没发货？】

话术：
"抱歉让您久等了
我帮您查一下订单状态
（查询后）
您的订单已经发货了，单号是XXX
预计X天能到
我发您物流信息"

【客户：快递破损了】

话术：
"非常抱歉！这种情况我们负责
您拍张照片发我
我这边给您补发或者退款
您看怎么处理方便？"
```

### 2. 产品问题

```
【客户：狗狗不爱吃】

话术：
"感谢您的反馈
狗狗不爱吃可能有几个原因：
1. 换粮太快，需要过渡
2. 口味不适应
3. 身体不舒服

建议：
- 逐步换粮，7天过渡
- 拌点零食/罐头引导
- 观察狗狗状态

如果实在不吃，可以申请退款
我们支持7天无理由退换"

【客户：吃了软便】

话术：
"软便可能有几个原因：
1. 换粮太快，肠胃不适应
2. 吃多了
3. 狗粮不适合

建议：
- 减少喂食量
- 加点益生菌调理
- 观察几天

如果持续软便，建议：
- 换回原来的狗粮
- 或者咨询宠物医生

我们支持退换，您看怎么处理？"
```

### 3. 客户投诉

```
【客户：你们的服务太差了！】

话术：
"非常抱歉给您带来不好的体验
请问具体是什么问题？
我这边马上处理

（了解情况后）
这个问题确实是我们做得不够好
我代表团队向您道歉
我们会：
1. 立即改进
2. 给您补偿（退款/优惠券/赠品）
3. 加强培训，避免再发生

感谢您的反馈，让我们知道哪里需要改进"
```

## 七、话术使用技巧

### 1. 基本原则

```
✅ 真诚：真实分享，不夸大
✅ 专业：用数据说话
✅ 共情：理解客户需求
✅ 耐心：详细解答问题
✅ 主动：引导客户下单
```

### 2. 沟通技巧

```
倾听：先了解客户需求
共情：理解客户顾虑
引导：给出解决方案
促成：引导客户行动
跟进：确保客户满意
```

### 3. 注意事项

```
❌ 不要强行推销
❌ 不要贬低竞品
❌ 不要过度承诺
❌ 不要忽视客户感受
❌ 不要使用绝对化用语
```
