## 七、脚本创作要点

### 1. 黄金3秒法则

```
前3秒必须有钩子：
├── 提出问题："你家边牧是不是也这样？"
├── 制造悬念："辞职做宠物寄养，我后悔了吗？"
├── 展示痛点："边牧掉毛严重怎么办？"
├── 惊人数据："养狗5年，我花了10万"
└── 强烈对比："养狗前 vs 养狗后"
```

### 2. 内容结构

```
开头（3秒）：钩子，吸引注意力
中间（主体）：干货/故事/展示
结尾（5秒）：引导互动，关注/点赞/评论
```

### 3. 互动引导

```
├── "评论区告诉我你家狗子的情况"
├── "觉得有用的点个赞"
├── "关注我，分享更多养狗干货"
├── "需要推荐的私信我"
└── "转发给你养狗的朋友"
```
