### 2. 灯光布置

```
【三点布光法】

        主灯（45度角，高亮度）
              ↓
    ┌───────────────────────┐
    │                       │
    │        👤 主体        │
    │                       │
    └───────────────────────┘
         ↑            ↑
    辅灯（补阴影）   轮廓灯（分离背景）

【灯光比例】
├── 主灯：100%亮度
├── 辅灯：50-70%亮度
└── 轮廓灯：30-50%亮度
```

## 六、直播软件设置

### 1. 抖音直播设置

```
【画质设置】
├── 分辨率：1080P
├── 帧率：30fps
├── 码率：4000-6000kbps
└── 编码：H.264

├── 磨皮：30-40%

└── 大眼：10-20%

【滤镜设置】
└── 选择自然、清新的滤镜
```

## 七、设备采购建议

### 1. 采购顺序

```
第一阶段（必备）：
├── 手机支架
├── 补光灯
└── 领夹麦克风

第二阶段（升级）：
├── 稳定器
├── 无线麦克风
└── 灯光组

第三阶段（专业）：
├── 相机
├── 专业灯光
└── 监视器等
```

### 2. 采购渠道

| 渠道 | 优势 | 劣势 |
|------|------|------|
| 淘宝/京东 | 品类全、售后好 | 价格略高 |
| 拼多多 | 价格便宜 | 质量参差 |
| 二手平台 | 性价比高 | 需验货 |

### 3. 注意事项

```
采购前：
├── 明确需求，不盲目追求高端
├── 对比多个平台价格
├── 查看用户评价
└── 确认售后政策

采购后：
├── 及时测试设备
├── 熟悉设备操作
├── 保留购买凭证
└── 有问题及时退换
```
