## 三、回本周期测算

### 1. 寄养模式（理想状态）

```
收入测算：
- 寄养容量：10只狗
- 平均入住率：50%（淡季30%，旺季80%）
- 单价：100元/天
- 月收入：10只 × 50% × 100元 × 30天 = 15000元

支出测算：
- 房租：1000元/月
- 水电：500元/月
- 清洁用品：400元/月
- 推广：500元/月
- 合计：2400元/月

月利润：12600元
回本周期：3-5万元 ÷ 12600元 ≈ 2-4个月（理想状态）
```

### 2. 现实情况

- 前3个月入住率可能只有20-30%
- **实际回本周期：8-12个月**

### 3. 电商模式

```
收入测算：
- 月销100单
- 客单价：100元
- 利润率：30%
- 月利润：100 × 100 × 30% = 3000元

回本周期：1-2万元 ÷ 3000元 ≈ 3-6个月
```

## 四、资金规划建议

### 1. 资金分配原则

```
├── 启动资金：占总预算60%
├── 运营资金：占总预算30%
└── 应急资金：占总预算10%
```

### 2. 风险控制

- 不要一次性投入全部资金
- 预留至少3个月的运营资金
- 设置止损线，及时调整策略

## 五、成本优化建议

| 项目 | 优化方法 |
|------|----------|
| 场地 | 选择性价比高的位置，可考虑合租 |
| 装修 | 简约实用，避免过度装修 |
| 设备 | 二手设备可降低成本 |
| 货源 | 代理拿货，降低进货成本 |
| 推广 | 以内容营销为主，降低付费推广 |
