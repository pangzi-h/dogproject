# 第一批客户获取策略

> 专门针对启动初期（前30天）的获客执行方案，目标是快速获得前5-10个客户。

---

## 一、获客优先级

启动初期的获客渠道，按优先级排序：

| 优先级 | 渠道 | 转化难度 | 获客周期 | 推荐指数 |
|--------|------|----------|----------|----------|
| 1 | 熟人圈 | 极低 | 1-3天 | ⭐⭐⭐⭐⭐ |
| 2 | 朋友圈/私域 | 低 | 3-7天 | ⭐⭐⭐⭐⭐ |
| 3 | 小红书 | 中 | 7-14天 | ⭐⭐⭐⭐ |
| 4 | 社区宠物群 | 中 | 7-14天 | ⭐⭐⭐⭐ |

| 6 | 抖音 | 高 | 14-30天 | ⭐⭐ |

## 二、熟人圈获客（最优先）

### 为什么从熟人圈开始？

1. **信任成本最低**：不需要证明自己靠谱
2. **转化最快**：一个电话/微信就能成交
3. **可收集反馈**：完善服务流程
4. **可积累案例**：为后续推广提供素材

### 熟人圈梳理

**立即列出你的熟人资源：**

```
第一圈层（家人/发小/闺蜜）：
- 有养狗的亲戚：____________
- 有养狗的发小：____________
- 有养狗的闺蜜/兄弟：____________

第二圈层（同事/同学）：
- 现在的同事（养狗的）：____________
- 前同事（养狗的）：____________
- 同学（养狗的）：____________

第三圈层（朋友的朋友）：
- 群里有养狗的：____________
- 朋友圈经常晒狗的：____________
```

### 熟人推广话术

**话术1（直接告知）：**

```
XX，告诉你个事儿~
我开始做宠物寄养了，就在XX那边，
有小院，环境不错，还有监控可以随时看。
你家狗如果需要寄养，找我哈，
给你亲情价~
```

**话术2（顺便提及）：**

```
最近我在弄宠物寄养，
我家那两只边牧天天陪客人的狗玩[呲牙]
对了，你家狗有时候需要寄养不？
我这边给你留着位置~
```

**话术3（晒照片引发）：**
发朋友圈晒场地/边牧照片，配文：
"我家边牧又有新朋友来玩了~"
等熟人评论/点赞，私信跟进。

### 熟人优惠政策

| 客户类型 | 优惠 | 目的 |
|----------|------|------|
| 第一批熟人客户 | 8折 | 快速成交、积累案例 |
| 熟人推荐客户 | 首单9折 | 口碑传播 |

## 三、朋友圈获客（重点）

### 朋友圈推广策略

**发布时间：**

- 早上 8:00-9:00（通勤时段）
- 中午 12:00-13:00（午休时段）
- 晚上 20:00-21:00（黄金时段）

**发布节奏：**

| 阶段 | 频率 | 内容重点 |
|------|------|----------|
| 启动前一周 | 每天1-2条 | 边牧日常、场地准备 |
| 启动后一周 | 每天2-3条 | 服务展示、客户反馈 |
| 稳定期 | 每天1-2条 | 持续曝光、优惠活动 |

### 朋友圈发布计划（第1周）

**Day 1：预告**

```
配图：边牧照片或场地照片
文案：即将开启一个小事业，
边牧小院宠物寄养~
两只边牧当"店长"，欢迎来撩[呲牙]
坐标：郑州XX区
```

**Day 2：场地展示**

```
配图：场地环境照片（3-5张）
文案：小院收拾好了，
Independent空间，分区管理，
监控24小时开着。
有需要的可以约来看场地~
```

**Day 3：边牧日常+软广**

```
配图：边牧玩耍照片
文案：我家两只边牧又拆家了...
不过有新朋友来它们就很乖~
宠物寄养，找靠谱的找我[机智]
```

**Day 4：服务优势**

```
配图：服务内容图或文字图
文案：我们家寄养不一样：
✅ 早晚遛狗各30分钟
✅ 24小时监控随时看
✅ 每日状态反馈
✅ 专业铲屎官5年经验
坐标郑州XX区，欢迎咨询~
```

**Day 5：促销活动**

```
配图：边牧照片+优惠信息
文案：开业福利！
新客首单9折，熟人8折[礼花]
位置有限，需要提前约~
私信我哈
```

**Day 6：客户反馈（如有）**

```
配图：狗狗照片+客户好评截图
文案：第一只狗狗今天退房啦~
主人说很放心，下次还来[爱心]
感谢信任！
```

**Day 7：日常+互动**

```
配图：边牧日常
文案：周末遛狗[狗头]
你们家狗周末都干啥？
评论区聊聊~
（评论区：需要寄养的私信我哈）
```

## 四、小红书获客（持续）

### 冷启动策略

**前7天：**

- 每天发1-2条边牧日常
- 内容要真实、有趣
- 不要硬广，先积累粉丝

**8-14天：**

- 加入寄养服务介绍内容
- 展示场地环境
- 引导私信咨询

**14天后：**

- 发布客户案例（需主人同意）
- 适度促销引流
- 稳定持续输出

### 第一批获客内容重点

**获客型内容选题：**

| 内容类型 | 标题示例 | 目的 |
|----------|----------|------|
| 场地展示 | 郑州宠物寄养，小院是这样的 | 展示环境 |
| 服务介绍 | 家庭寄养vs宠物店，差在哪 | 差异化 |
| 真实案例 | 这只狗狗寄养了7天... | 建立信任 |
| 边牧日常 | 养边牧5年...（植入服务） | 吸粉+软广 |

### 私信转化流程

```
用户咨询 → 第一条回复（热情介绍）
    ↓
发送场地照片 → 询问宠物信息
    ↓
加微信详聊 → 发位置/报价
    ↓
预约参观 → 签约成交
```

**私信回复模板：**

```
第一条（介绍）：
您好！感谢关注~
我们是家庭式宠物寄养，在郑州XX区，
有独立场地，24小时监控。
方便加微信聊吗？我发您场地照片~
```

```
第二条（加微信后）：
[发送场地照片3-5张]
这是我们的场地环境~
您家狗狗是什么品种？
打算什么时候寄养呢？
```

## 五、社区群获客

### 寻找目标群

**方法1：问遛狗的人**

```
话术："您好，请问这附近有宠物群吗？
我家两只边牧想找狗玩~"
```

**方法2：问物业**

```
话术："您好，请问咱们小区有宠物交流群吗？"
```

**方法3：搜索**

```
- 搜索微信群：小区名+宠物
- 业主群里的宠物话题
- 附近的遛狗群
```

### 进群策略

**前7天：不广告，先混脸熟**

- [ ] 进群后自我介绍（简单说养边牧）
- [ ] 回答别人的养宠问题
- [ ] 分享边牧照片（引起注意）
- [ ] 点赞互动，建立好感

**话术示例：**

```
大家好，我也养狗，两只边牧，养了5年~
以后多交流[呲牙]
```

**第7天后：适度推广**

**触发式推广（有人问寄养时）：**

```
当群里有人问：
"有没有靠谱的寄养推荐？"

回复：
"我刚好知道一家，就在XX那边，
小院式寄养挺好的，
我朋友去过，可以推给你~"
（如果是自己的，可以说：
"我这边也做寄养，您可以过来看看环境~"）
```

**主动推广话术（适度）：**

```
坐标郑州XX区，家庭式宠物寄养~
两只边牧当店长，24小时监控，
有需要的可以私信看环境[呲牙]
```

## 九、获客常见问题

### Q1：没有客户怎么办？

**分析原因：**

1. 曝光不够 → 加大朋友圈、小红书发布
2. 内容没吸引力 → 优化内容和照片
3. 价格问题 → 适度调整或增加促销
4. 信任问题 → 提供看场地、先试住

### Q2：客户嫌贵怎么办？

**应对话术：**

```
"确实比一些宠物店稍贵一点，
但我们是小院式寄养，
笼子大、活动空间大，
遛狗时间是宠物店的两倍以上，
性价比其实更高的~
第一次来给您打个折？"
```

### Q3：客户担心不靠谱怎么办？

**应对措施：**

```
1. 邀请实地参观
2. 展示监控画面
3. 介绍服务流程
4. 展示其他客户好评
5. 提供试住（半天到一天）
```
