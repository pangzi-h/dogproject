# 寄养服务定价策略

## 一、定价原则

### 1. 三大导向

```
成本导向 + 竞争导向 + 价值导向
```

### 2. 成本核算

```
成本计算（狗粮由主人自带）：
├── 房租成本：1000元/月 ÷ 30天 = 33元/天
├── 人工成本：2人 × 5000元/月 ÷ 30天 ÷ 10只 = 33元/只/天
├── 水电成本：5元/只/天
├── 清洁用品：3元/只/天
├── 其他成本：4元/只/天
└── 合计成本：78元/只/天

最低定价：80-90元/天（保证基本利润）

注：狗粮由主人自带，降低运营成本，也避免狗狗换粮不适
```

### 3. 狗粮相关说明

```
【为什么要求自带狗粮？】

✅ 避免肠胃不适
   - 突然换粮容易引起软便、拉稀
   - 保持狗狗饮食习惯稳定

✅ 减少过敏风险
   - 每只狗狗体质不同
   - 继续吃熟悉的狗粮更安全

✅ 降低寄养成本
   - 主人自带狗粮，寄养费用更低
   - 性价比更高

【忘记带狗粮怎么办？】
- 可提供应急狗粮：10-20元/餐
- 可代购主人指定品牌狗粮（按实际价格）
- 推荐使用伊芙琳狗粮（代理价优惠）
```

## 六、定价注意事项

### 1. 明码标价

- 所有价格公开透明
- 服务内容清晰说明
- 无隐藏收费

### 2. 灵活调整

- 根据市场反馈调整
- 参考竞争对手定价
- 考虑成本变化

### 3. 价值传递

- 让客户感受到物有所值
- 强调服务差异化
- 展示客户好评
