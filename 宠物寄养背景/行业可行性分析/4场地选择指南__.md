# 场地选择指南

## 一、选址原则

### 1. 优先考虑

```
✅ 靠近中高端小区
   └── 养宠密度高，消费能力强

✅ 交通便利
   └── 方便客户接送

✅ 有独立院落或大阳台
   └── 遛狗空间充足

✅ 通风良好
   └── 避免异味投诉
```

### 2. 避免选择

```
❌ 纯住宅区
   └── 容易扰民投诉

❌ 偏远地区
   └── 客户接送不便

❌ 面积过小
   └── 无法扩展

❌ 周边环境差
   └── 影响客户体验
```

---

## 二、场地要求

### 1. 面积规划

| 规模 | 面积 | 容量 | 适用阶段 |
|------|------|------|----------|
| 小型 | 30-50㎡ | 5-8只狗 | 初期试水 |
| 中型 | 80-100㎡ | 15-20只狗 | 稳定运营 |
| 大型 | 150㎡以上 | 30只以上 | 规模化 |

### 2. 分区设计

```
场地布局：

┌─────────────────────────────────────────┐
│                                         │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│  │ 寄养区  │  │ 活动区  │  │ 洗护区  │ │
│  │         │  │         │  │         │ │
│  │ 笼舍    │  │ 室内玩耍│  │ 洗澡美容│ │
│  │ 储物    │  │ 遛狗区  │  │ 吹干    │ │
│  └─────────┘  └─────────┘  └─────────┘ │
│                                         │
│  ┌─────────────────────────────────────┐│
│  │            隔离区                   ││
│  │   （新入住/生病狗狗隔离观察）       ││
│  └─────────────────────────────────────┘│
│                                         │
└─────────────────────────────────────────┘
```

### 3. 设施要求

| 设施 | 要求 | 说明 |
|------|------|------|
| 笼舍 | 独立、通风、易清洁 | 每只狗独立空间 |
| 空调 | 必备 | 夏季降温 |
| 监控 | 24小时 | 客户可远程查看 |
| 消毒设备 | 紫外线灯、消毒液 | 定期消毒 |
| 通风系统 | 良好 | 避免异味 |
| 防护设施 | 围栏、门禁 | 防止逃跑 |

## 三、选址流程

### 1. 信息收集

```
渠道：
├── 58同城、安居客等租房平台
├── 本地房产中介
├── 朋友圈、社群打听
└── 实地走访目标区域
```

### 2. 实地考察

| 考察项 | 要点 |
|--------|------|
| 位置 | 交通是否便利 |
| 周边 | 是否有小区、宠物店 |
| 环境 | 是否干净、安静 |
| 房东 | 是否支持做宠物寄养 |
| 证件 | 是否能办理相关证照 |

### 3. 对比评估

| 场地 | 租金 | 面积 | 位置 | 交通 | 综合评分 |
|------|------|------|------|------|----------|
| 场地A | 1万/年 | 50㎡ | ★★★★ | ★★★ | 8分 |
| 场地B | 1.2万/年 | 60㎡ | ★★★ | ★★★★ | 8分 |
| 场地C | 1.3万/年 | 80㎡ | ★★★ | ★★★ | 7分 |

## 四、郑州推荐区域

### 1. 优先考虑区域

| 区域 | 优势 | 适合程度 |
|------|------|----------|
| 金水区 | 消费能力强、养宠密度高 | ⭐⭐⭐⭐⭐ |
| 郑东新区 | 高端小区多 | ⭐⭐⭐⭐⭐ |
| 二七区 | 交通便利 | ⭐⭐⭐⭐ |
| 管城区 | 租金相对便宜 | ⭐⭐⭐ |

### 2. 具体位置建议

- 靠近大型社区（如建业、万科等）
- 商住两用区域
- 有停车位的场地

---

## 五、注意事项

### 1. 合同签订

```
必看条款：
├── 租期及续租条件
├── 租金及支付方式
├── 是否允许做宠物寄养
├── 装修改造权限
├── 水电费用标准
├── 违约责任
└── 退租条件
```

### 2. 证照办理

| 证照 | 办理部门 | 备注 |
|------|----------|------|
| 营业执照 | 市场监督管理局 | 个体户即可 |
| 动物防疫条件合格证 | 农业农村局 | 部分地区需要 |

### 3. 邻里关系

- 入住前与邻居打好招呼
- 做好隔音措施
- 保持环境清洁
- 避免噪音扰民
