## 二、遛狗操作规范

### 准备物品

- [ ] 牵引绳（检查有无破损）
- [ ] 捡屎袋
- [ ] 手机（看时间、拍照）

### 遛狗流程

```
步骤1：出门前检查
□ 确认牵引绳系牢
□ 检查项圈是否合适
□ 确认狗状态正常

步骤2：出门遛狗
□ 关好门（防止其他狗跑出）
□ 选择固定路线
□ 控制时间（30分钟左右）
□ 让狗充分活动

步骤3：遛狗过程
□ 观察狗的精神状态
□ 避免接触陌生狗（除非确定友好）
□ 捡屎、观察大便状态
□ 避免狗吃地上的东西

步骤4：返回
□ 擦脚（如需要）
□ 擦身体（如下雨/泥泞）
□ 放回笼位
□ 补水
□ 记录排便情况
```

## 三、喂食操作规范

### 喂食前检查

- [ ] 确认该狗的饮食要求（主人家嘱咐）
- [ ] 确认喂食量
- [ ] 确认喂食时间
- [ ] 检查狗粮是否新鲜

### 喂食流程

```
步骤1：准备食物
□ 按量称取狗粮
□ 如需加热，适当加热
□ 准备新鲜饮用水

步骤2：喂食
□ 将食盆放入笼内
□ 观察进食情况
□ 记录进食量

步骤3：喂食后
□ 15-30分钟后检查
□ 如有剩余，记录原因
□ 清洗食盆（下一餐前）
□ 换新鲜饮用水
```

## 四、环境清洁标准

### 每日清洁（必做）

**早晨清洁：**

- [ ] 清扫所有笼子内部
- [ ] 更换脏的尿垫
- [ ] 拖地（消毒液配比）
- [ ] 倒垃圾

**下午清洁：**

- [ ] 检查尿垫，更换脏的
- [ ] 局部清扫
- [ ] 拖地（如需要）

### 每周深度清洁

**周一：**

- [ ] 全面大扫除
- [ ] 所有笼子深度清洁
- [ ] 紫外线消毒1小时
- [ ] 清洗所有垫子
- [ ] 清洗玩具

### 消毒液配比

| 消毒液类型 | 配比 | 用途 |
|------------|------|------|
| 宠物专用消毒液 | 1:100 | 地面、笼子日常消毒 |
| 84消毒液 | 1:200 | 深度消毒（需通风后使用） |
| 紫外线灯 | 1小时/次 | 空间消毒（人宠离开） |

## 七、应急处理流程

### 常见问题处理

**问题1：拉稀**

```
轻度（1-2次，精神好）：
1. 禁食6-12小时，只给水
2. 喂益生菌
3. 观察
4. 通知主人

中度（3次以上或带血）：
1. 拍照记录
2. 通知主人
3. 准备送医
4. 保留样本
```

**问题2：不吃东西**

```
一顿不吃：
1. 观察
2. 尝试换口味
3. 通知主人

24小时不吃：
1. 检查口腔
2. 测体温
3. 通知主人商量送医
```

**问题3：打架**

```
发生打架时：
1. 立即分开（用水泼/板子隔开）
2. 检查伤情
3. 轻伤自行处理，重伤送医
4. 通知主人
5. 记录过程
```

**问题4：邻居投诉**

```
发生投诉时：
1. 态度诚恳接受
2. 了解具体问题
3. 留下联系方式
4. 采取改进措施
5. 事后沟通感谢
```
