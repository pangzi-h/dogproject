## 三、拍摄执行方案

### 拍摄设备

**基础设备：**

- 手机（iPhone或安卓旗舰）
- 三脚架（稳定拍摄）
- 补光灯（光线不足时）

### 拍摄场景

| 场景 | 拍摄内容 | 光线要求 | 注意事项 |
|------|----------|----------|----------|
| 室内日常 | 边牧吃饭、睡觉、玩耍 | 自然光充足 | 抓拍自然状态 |
| 户外遛狗 | 边牧奔跑、互动 | 避免逆光 | 动态抓拍 |
| 寄养场地 | 场地展示、服务展示 | 白天光线好 | 保持整洁 |

### 拍摄技巧

**照片拍摄：**

```
1. 抓拍自然状态，不摆拍
2. 光线要充足，避免逆光
3. 对焦在狗的眼睛
4. 多拍几张，后期选最好的
5. 适当蹲下，和狗平视
```

**视频拍摄：**

```
1. 保持稳定（用三脚架或稳定器）
2. 横屏拍摄（16:9）
3. 光线要充足
4. 拍摄时长15-60秒
5. 多角度拍摄
```

### 每日拍摄计划

| 时间 | 拍摄内容 | 用途 |
|------|----------|------|
| 早晨遛狗 | 边牧状态、活动 | 日常素材 |
| 喂食时间 | 吃饭状态 | 日常/寄养素材 |
| 下午活动 | 玩耍互动 | 日常素材 |
| 寄养过程 | 狗狗状态（需主人同意） | 寄养素材 |

## 四、文案撰写模板

### 边牧日常类

**标题公式：** [场景]+[趣味/悬念]

**正文模板：**

```
[场景描述]（2-3句）
[吐槽/感叹]（1-2句）
[互动问题]（1句）
#标签
```

**示例：**

```
标题：养边牧5年，我终于放弃了教它捡球...

每次扔出去，它就看着我，
眼神仿佛在说：你自己扔的自己捡。
教了5年，最后捡球的是我。
养边牧的都懂...[捂脸]

你们家狗会捡球吗？

#边牧 #养狗日记 #边牧日常
```

### 养宠干货类

**标题公式：** [数字]+[痛点] 或 [痛点]+[解决方案]

**正文模板：**

```
[开头发问/场景引入]
[分点叙述]
  1️⃣ __________
  2️⃣ __________
  3️⃣ __________
[结尾互动]
#标签
```

**示例：**

```
标题：养狗5年，我踩过的3个坑

今天想分享几个养狗容易踩的坑，
新手铲屎官一定要避开！

1️⃣ 狗粮不要随便换
狗狗肠胃很敏感，换粮要慢慢过渡，
不然容易拉稀！

2️⃣ 打疫苗不是越多越好
按照规范来，不要过度疫苗。
我家两只边牧都是按医生建议打的。

3️⃣ 别给狗吃人吃的
巧克力、葡萄、洋葱...很多对狗有毒！
哪怕它用小狗眼神看你也不行！

还有啥想问的评论区见~

#养狗 #新手养狗 #狗狗
```

### 寄养介绍类

**标题公式：** [地点]+[服务]+[亮点]

**正文模板：**

```
[场景引入]
[服务亮点]
  📍地址：________
  📹监控：________
  🦮遛狗：________
  🍜喂养：________
[结尾引导]
#标签
```

**示例：**

```
标题：郑州宠物寄养，我们是这样做的

经常有人问：你们家寄养是啥样的？
今天带大家看看~

📍场地：独立小院，分区管理
📹监控：24小时监控，随时可看
🦮遛狗：早晚各一次，每次30分钟+
🍜喂养：按主人要求，可自带粮

每只狗都会用心照顾，
就像对自己家的一样。

坐标郑州XX区，有需要可以私信~

#郑州宠物寄养 #宠物寄养 #郑州
```

### 寄养故事类

**标题公式：** [悬念]+[故事]

**正文模板：**

```
[故事开头-引入]
[故事发展]
[故事结尾-感悟]
[软性引导]
#标签
```

**示例：**

```
标题：这只泰迪来寄养时，主人哭了...

今天来了一只小泰迪，叫豆豆。
主人送来的时候，看着豆豆进了笼子，
眼眶一下就红了。

"第一次把豆豆交给别人照顾..."
主人说。

我拍着她肩膀说：
"放心，我会每天发照片视频，
监控也开着，你随时能看到它。"

接下来的7天，每天更新豆豆状态，
主人从一开始的担心,
到后来笑着说"豆豆比在家还开心"。

离别时主人特意带了礼物来，
说下次还来。

做寄养，最感动的就是这种时刻。

#宠物寄养 #狗狗 #郑州
```

## 五、封面设计规范

### 封面尺寸

- 小红书：3:4竖图（1242×1660px）
- 抖音：9:16竖图（1080×1920px）

### 封面风格

**原则：**

1. 清晰明亮
2. 主体突出（狗在中心）
3. 文字简洁醒目
4. 色调统一（暖色系推荐）

### 封面工具

- 醒图APP（推荐）
- 黄油相机
- Canva
- 美图秀秀

### 封面模板

```
┌─────────────────────┐
│                     │
│                     │
│   [主体照片]         │
│                     │
│                     │
│                     │
│ ████████████████    │
│ [大字标题]           │
│ ████████████████    │
│ [小字说明]           │
└─────────────────────┘
```

**字体建议：**

- 标题：粗体、综艺体、黑体
- 正文：宋体、微软雅黑

## 七、内容优化方向

### 爆款特征分析

**什么样的内容容易爆：**

1. 有痛点（解决实际问题）
2. 有情绪（搞笑/温馨/感人都行）
3. 有争议（引发讨论）
4. 有价值（干货分享）

### 低流量内容分析

**什么样的内容流量差：**

1. 纯广告（无价值）
2. 图片质量差
3. 标题无聊
4. 内容平淡无特点

### 优化措施

**遇到流量差时：**

1. 分析标题是否够吸引
2. 分析封面是否够醒目
3. 分析内容是否有价值
4. 参考同类型爆款
5. 调整发布时间

---

> **内容创作要点：**
> 
> 1. 持续输出，不能断更
> 2. 真实第一，不装不演
> 3. 及时互动，回复评论
> 4. 分析数据，优化方向
> 5. 引导私信，转化客户
