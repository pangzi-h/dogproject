## 二、构图技巧

### 1. 基础构图法则

```
【三分法构图】
┌─────────┬─────────┬─────────┐
│         │         │         │
│    ●    │         │         │
│         │         │         │
├─────────┼─────────┼─────────┤
│         │    ×    │         │
│         │   主体  │         │
├─────────┼─────────┼─────────┤
│         │         │    ●    │
│         │         │         │
└─────────┴─────────┴─────────┘

× = 主体位置（交叉点）
● = 兴趣点

技巧：把边牧的眼睛放在交叉点上
```

### 2. 拍摄角度选择

| 角度 | 效果 | 适用场景 |
|------|------|----------|
| **平视** | 真实自然 | 日常记录、对话感 |
| **低角度** | 狗狗视角、可爱 | 边牧正面、奔跑 |
| **高角度** | 全景、小巧感 | 睡觉、吃饭全景 |
| **特写** | 细节、情感 | 眼神、毛发、表情 |

## 三、光线运用

### 1. 自然光拍摄

```
【最佳时间】
├── 黄金时段：日出后1小时、日落前1小时
│   └── 光线柔和，色调温暖
├── 上午：9-11点
│   └── 光线充足，适合户外
└── 下午：3-5点
    └── 光线柔和，适合室内

【避免时间】
└── 正午12-2点：光线太强，阴影重

【室内拍摄】
├── 靠近窗户，利用自然光
├── 避免逆光（背对窗户）
└── 使用补光灯填充阴影
```

### 2. 补光灯使用

```
【补光灯位置】

    │    🐕 主体    │  ← 45度角照射
    │               │
    └───────────────┘
         📱 手机

【补光强度】
├── 白天：补光灯亮度30-50%
```

## 四、运镜技巧

### 1. 基础运镜手法

```
【推镜头】
效果：从远到近，突出主体
手法：手机缓慢向前移动
用途：展示边牧表情、产品细节

【拉镜头】
效果：从近到远，展示环境
手法：手机缓慢向后移动
用途：展示场地、全景

【摇镜头】
效果：左右/上下移动，展示空间
手法：手机固定位置，转动拍摄
用途：展示场地、遛狗路线

【跟镜头】
效果：跟随主体移动
手法：手机跟随边牧移动
用途：遛狗、奔跑、玩耍

【环绕镜头】
效果：360度展示主体
手法：围绕边牧转圈拍摄
用途：展示边牧全貌、产品包装
```

### 2. 稳定技巧

```
【手持稳定】
├── 双手持机，手肘贴紧身体
├── 膝盖微弯，身体为稳定器
├── 移动时用"猫步"，脚跟到脚尖
└── 呼吸平稳，避免抖动

【使用稳定器】
├── 预算200-500元：大疆OM系列入门款
├── 设置：开启防抖模式
└── 练习：先练走直线，再练运镜
```

## 六、剪辑技巧

### 1. 剪辑软件推荐

| 软件 | 难度 | 功能 | 推荐 |
|------|------|------|------|
| **剪映** | 简单 | 基础剪辑+特效+字幕 | ⭐⭐⭐⭐⭐ |
| **CapCut** | 简单 | 剪映国际版，功能相同 | ⭐⭐⭐⭐⭐ |
| **必剪** | 简单 | B站出品，适合B站 | ⭐⭐⭐⭐ |
| **PR** | 专业 | 专业剪辑软件 | ⭐⭐⭐ |

### 2. 剪映基础操作

```
【剪辑流程】

1. 导入素材
   └── 选择拍摄好的视频片段

2. 粗剪
   ├── 删除废片
   ├── 保留精华片段
   └── 按脚本顺序排列

3. 精剪
   ├── 调整每个片段时长
   ├── 删除停顿、废话
   └── 控制总时长（30-60秒）

4. 添加元素
   ├── 转场：片段之间添加转场
   ├── 字幕：自动识别+手动修改
   ├── BGM：选择合适背景音乐
   ├── 音效：添加音效增强效果
   └── 滤镜：统一色调

5. 导出
   ├── 分辨率：1080P
   ├── 帧率：30fps
   └── 质量：高质量
```

### 3. 剪辑节奏技巧

```
【节奏控制】

快节奏（适合搞笑、日常）：
├── 每个镜头1-3秒
├── 快速切换
├── 配合快节奏BGM
└── 适合：边牧日常、搞笑视频

慢节奏（适合知识、情感）：
├── 每个镜头3-5秒
├── 平稳过渡
├── 配合舒缓BGM
└── 适合：知识分享、情感故事

【黄金法则】
├── 前3秒必须有钩子
├── 每5秒一个变化点
├── 删除所有无效镜头
└── 结尾引导互动
```
