# 短视频脚本模板大全

## 一、模板1：边牧日常类

```
【标题】：两只边牧的早晨，一个想遛一个想睡

【时长】：30-45秒

【脚本】：
┌──────────────────────────────────────────────────────┐
│ 0-3秒（钩子）                                        │
│ 画面：边牧A兴奋地叼着牵引绳，边牧B懒洋洋趴着         │
│ 配音/字幕："养两只边牧是什么体验？每天早上都是战争！" │
│                                                      │
│ 4-20秒（主体）                                       │
│ 画面：快剪对比                                       │
│ ├── 边牧A：疯狂转圈、叼绳子、扒门                    │
│ ├── 边牧B：翻白眼、打哈欠、继续睡                    │
│ └── 主人无奈的表情                                   │
│ 配音："老大恨不得住在户外，老二恨不得住在床上"       │
│                                                      │
│ 21-30秒（转折）                                      │
│ 画面：两只狗一起出门，都变得兴奋                     │
│ 配音："但一出门，懒狗秒变疯狗..."                    │
│                                                      │
│ 31-40秒（结尾）                                      │
│ 画面：两只狗在草地上奔跑                             │
│ 配音："这就是边牧，又爱又恨"                         │
│ 引导："你家狗子早上也是这样吗？评论区聊聊"           │
└──────────────────────────────────────────────────────┘

【BGM】：轻松欢快的音乐
【标签】：#边牧 #狗狗日常 #养狗
```

## 二、模板2：养狗知识类

```
【标题】：边牧掉毛严重？这3招亲测有效

【时长】：45-60秒

【脚本】：
┌──────────────────────────────────────────────────────┐
│ 0-3秒（钩子）                                        │
│ 画面：家里到处是狗毛，边牧正在掉毛                   │
│ 配音："养边牧5年，我家狗毛能织毛衣了"                │
│ 字幕：掉毛季的绝望                                   │
│                                                      │
│ 4-10秒（痛点放大）                                   │
│ 画面：快速展示各处的狗毛                             │
│ ├── 沙发上的毛                                       │
│ ├── 衣服上的毛                                       │
│ └── 空气中飘的毛                                     │
│ 配音："每天吸尘器吸三遍，还是到处都是"               │
│                                                      │
│ 11-40秒（解决方案）                                  │
│ 画面：分三段展示                                     │
│                                                      │
│ 【第1招：梳毛工具】                                  │
│ 画面：展示梳毛工具，演示梳毛                         │
│ 配音："这种针梳，每天梳10分钟，能梳下一只小狗的毛量" │
│ 字幕：针梳/褪毛梳 10分钟/天                          │
│                                                      │
│ 【第2招：饮食调整】                                  │
│ 画面：展示狗粮，边牧吃狗粮                           │
│ 配音："换了对毛发好的狗粮，掉毛量直接减半"           │
│ 字幕：选择含Omega-3的狗粮                            │
│                                                      │
│ 【第3招：补充营养】                                  │
│ 画面：展示鱼油/蛋黄                                  │
│ 配音："每周2-3个蛋黄，或者鱼油，毛发亮泽掉毛少"      │
│ 字幕：蛋黄2-3个/周 或 鱼油                           │
│                                                      │
│ 41-50秒（效果展示）                                  │
│ 画面：对比图，掉毛前后对比                           │
│ 配音："坚持一个月，效果肉眼可见"                     │
│                                                      │
│ 51-60秒（结尾引导）                                  │
│ 画面：边牧毛发顺滑的特写                             │
│ 配音："关注我，分享更多养边牧干货"                   │
│ 引导："需要狗粮推荐的，评论区扣1"                    │
└──────────────────────────────────────────────────────┘

【BGM】：轻快节奏感强的音乐
【标签】：#边牧掉毛 #养狗知识 #狗狗护理
```

## 三、模板3：产品测评类

```
【标题】：这款狗粮我喂了3个月，真实感受

【时长】：60-90秒

【脚本】：
┌──────────────────────────────────────────────────────┐
│ 0-5秒（钩子）                                        │
│ 画面：边牧吃狗粮的特写                               │
│ 配音："伊芙琳这款狗粮，我家两只边牧吃了3个月"        │
│ 字幕：真实测评 不吹不黑                              │
│                                                      │
│ 6-15秒（背景介绍）                                   │
│ 画面：展示狗粮包装、配料表                           │
│ 配音："先说价格，我拿的代理价，比某宝便宜30%"        │
│ 字幕：代理价 vs 零售价对比                           │
│                                                      │
│ 16-30秒（配料分析）                                  │
│ 画面：配料表特写，逐条展示                           │
│ 配音："看配料表，前三位是鸡肉、鱼肉、牛肉"           │
│ 字幕：肉含量65% 无谷物 无诱食剂                       │
│ 配音："无谷配方，对边牧这种容易过敏的狗很友好"       │
│                                                      │
│ 31-50秒（实际效果）                                  │
│ 画面：展示边牧状态                                   │
│ ├── 毛发顺滑                                         │
│ ├── 便便正常                                         │
│ └── 精神状态好                                       │
│ 配音："吃了3个月，最明显的是毛发变亮了"              │
│ 配音："便便成型不软便，说明吸收好"                   │
│ 字幕：3个月真实效果                                  │
│                                                      │
│ 51-70秒（优缺点总结）                                │
│ 画面：文字总结                                       │
│ 配音："说下缺点，颗粒稍大，小狗可能不太适合"         │
│ 字幕：                                               │
│ ✅ 优点：性价比高、毛发亮、便便好                    │
│ ❌ 缺点：颗粒稍大、包装普通                          │
│                                                      │
│ 71-90秒（结尾引导）                                  │
│ 画面：边牧吃完舔盘子的镜头                           │
│ 配音："总体来说，这个价位性价比很高"                │
│ 引导："想了解价格的，私信我或评论区留言"             │
└──────────────────────────────────────────────────────┘

【BGM】：轻松背景音乐
【标签】：#狗粮测评 #伊芙琳 #宠物用品
```
